# Davidson RMP Overlay

Load this as an unpacked Chrome extension.

## Install
1. Open `chrome://extensions`
2. Turn on **Developer mode**
3. Click **Load unpacked**
4. Select this folder
5. Refresh `https://course-schedule.davidson.edu/#/`

## Notes
- Works by finding the instructor column in the Davidson schedule table and replacing instructor names with clickable links.
- Looks up professors live from Rate My Professors for **Davidson College**.
- Results are cached in Chrome storage for 3 days.
