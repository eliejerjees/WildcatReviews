Privacy Policy

WildcatReviews Privacy Policy

Effective date: March 16, 2026

WildcatReviews is a Chrome extension that displays professor review information on supported Davidson College pages. This Privacy Policy explains what data the extension accesses, how that data is used, and what is stored.

1. What WildcatReviews Does

WildcatReviews helps users view professor rating information directly on supported academic pages. Its purpose is limited to identifying instructor information on those pages and showing related review data in context.

2. Information the Extension Accesses

The extension may access limited website content on supported pages, such as:

  • Professor names
  • Course or section information
  • Related page text needed to match instructors with reviews

This access is used only to provide the extension’s core functionality.

3. How Information Is Used

Information accessed by WildcatReviews is used only to:

  • Identify the instructor shown on a supported page
  • Fetch matching professor review data
  • Display that review data to the user inside the page

WildcatReviews does not use this information for advertising, profiling, or unrelated analytics.

4. Local Storage

WildcatReviews may use browser storage, such as chrome.storage, to store:

  • Cached professor rating results
  • Extension preferences or settings

This local storage is used to improve speed, reduce repeated requests, and support normal extension functionality.

5. Data Sharing

WildcatReviews does not sell user data.

WildcatReviews does not transfer user data to third parties except when necessary to retrieve professor rating information required for the extension’s single purpose.

6. Remote Code

WildcatReviews does not use remote code. All executable code is packaged with the extension.

7. Data WildcatReviews Does Not Collect

WildcatReviews is not designed to collect or store:

  • Passwords or authentication credentials
  • Payment or financial information
  • Health information
  • Personal communications
  • Precise location data

8. Changes to This Policy

This Privacy Policy may be updated from time to time. Any changes will be posted on this page with an updated effective date.
